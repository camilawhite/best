# 🛡️ 이은성 고객님 맞춤형 보장 분석 리포트

대한민국 통계청 데이터와 최신 의학 트렌드를 기반으로 한 전문 보험 보장 분석 리포트입니다. 고객님의 현재 보장 현황을 분석하고, 발생 가능한 사각지대를 시각화하여 최적의 솔루션을 제안합니다.

## ✨ 주요 특징

- **데이터 기반 분석**: 통계청 2024년 사망원인통계 및 최신 항암 치료 비용 데이터를 시각화하여 객관적인 분석 제공.
- **모바일 최적화 (Mobile-First)**: 스마트폰에서 앱처럼 부드럽게 작동하는 반응형 디자인 및 스와이프 제스처 지원.
- **인터랙티브 차트**: Recharts를 활용한 직관적인 막대 그래프 및 파이 차트로 보장 공백을 한눈에 파악.
- **전문적인 디자인**: Tailwind CSS와 Framer Motion을 사용한 세련된 애니메이션과 UI/UX.
- **단일 파일 배포**: 외부 라이브러리를 CDN 방식으로 포함하여 `index.html` 파일 하나로 즉시 배포 및 실행 가능.

---

## 📊 리포트 구성 내용

1. **사망 원인 분석**: 대한민국 3대 사인(암, 심장, 뇌)의 비중 확인.
2. **암 보장 분석**: 단순 진단비를 넘어선 고가의 최신 항암 치료비(표적항암, 로봇수술 등) 대비 필요성.
3. **심장 질환 사각지대**: 급성심근경색만으로는 부족한 '협심증' 보장 범위 분석.
4. **뇌혈관 질환 골든타임**: 터지기 전 발견하는 '뇌동맥류' 보장의 중요성.
5. **100% 안심 솔루션**: 조기 발견과 완치를 돕는 현대적 보장 패러다임 제안.

## 🛠 사용된 기술 (Tech Stack)

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Charts**: Recharts
- **Icons**: Lucide React
- **Deployment**: GitHub Pages

## 🚀 설치 및 배포 방법

이 프로젝트는 GitHub Pages에 최적화된 **단일 HTML 파일** 구조입니다.

1. 이 저장소의 `index.html` 파일을 다운로드합니다.
2. 본인의 GitHub 저장소에 업로드합니다.
3. 저장소 설정(Settings) > Pages에서 배포 브랜치를 선택하면 즉시 웹사이트가 활성화됩니다.
